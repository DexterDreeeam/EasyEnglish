# Utils Module — Interface

> **Status:** **frozen** (since iter-015). Any breaking change to anything in
> `## Public` requires an ADR.

Crate name: `ee-utils`.

## Public

### `DynamicResult<T>` (consumer-side handle)

```rust
pub struct DynamicResult<T: Send + Sync + 'static> { /* opaque */ }

impl<T: Send + Sync + 'static> DynamicResult<T> {
    /// Construct. Returns `(consumer, producer)` sharing one allocation.
    pub fn new(initial: T) -> (Self, DynamicResultProducer<T>);

    /// Snapshot clone of the current state. Acquires a read lock for the
    /// duration of the clone.
    pub fn get(&self) -> T
    where
        T: Clone;

    /// Notify the producer the result is no longer needed. Idempotent.
    /// Wakes any waiters on `wait` and `producer.wait_for_cancel`.
    pub fn cancel(&self);

    /// Block until: the producer mutates state (`Signal::Changed`), the
    /// result terminates (`Signal::Finished` or `Signal::Failed`), or
    /// `timeout` elapses (`Signal::TimedOut`). `None` timeout = wait
    /// indefinitely (never returns `TimedOut`).
    pub fn wait(&self, timeout: Option<Duration>) -> Signal;
}

impl<T: Send + Sync + 'static> Drop for DynamicResult<T> {
    /// RAII: calls `cancel()`.
    fn drop(&mut self) { self.cancel(); }
}
```

### `DynamicResultProducer<T>` (producer-side handle)

```rust
pub struct DynamicResultProducer<T: Send + Sync + 'static> { /* opaque */ }

impl<T: Send + Sync + 'static> DynamicResultProducer<T> {
    /// Mutate the state under a write lock. Always runs `f`. Always wakes
    /// waiters. Behaviour after the result is finished: the closure still
    /// runs (state is updated for any subsequent `get`), but no consumer is
    /// waiting to observe the wake.
    pub fn update(&self, f: impl FnOnce(&mut T));

    /// Mark the result completed. Idempotent. No-op if already terminated
    /// (cancel or fail won the race).
    pub fn complete(&self);

    /// Mark the result failed with `reason`. Idempotent. No-op if already
    /// terminated.
    pub fn fail(&self, reason: impl Into<String>);

    /// Block until cancellation is observed, the result otherwise terminates,
    /// or `timeout` elapses. Returns `true` only on cancellation. `None`
    /// timeout = wait indefinitely.
    pub fn wait_for_cancel(&self, timeout: Option<Duration>) -> bool;

    /// Spawn a worker thread running `f`. If `f` panics, the panic message
    /// is converted to a string and forwarded via `fail()`. Otherwise the
    /// producer drops at the end of `f`, which calls `complete()`.
    pub fn spawn<F>(self, f: F)
    where
        F: FnOnce(&Self) + Send + 'static;
}

impl<T: Send + Sync + 'static> Drop for DynamicResultProducer<T> {
    /// RAII: calls `complete()`. Guarantees `Finished` to consumers waiting
    /// on `wait` even if the worker panics outside of `spawn`'s catcher.
    fn drop(&mut self) { self.complete(); }
}
```

### `Signal`

```rust
#[derive(Debug, Clone)]
pub enum Signal {
    Changed,         // state mutated since last wake
    Finished,        // cancel OR complete (terminal)
    Failed(String),  // fail() called (terminal)
    TimedOut,        // wait's timeout expired
}
```

## Terminal-state decision table

Whichever transition fires **first** wins. Subsequent calls are no-ops.

| First to fire | `wait` returns |
|---|---|
| `cancel()` (incl. `DynamicResult::Drop`) | `Finished` |
| `complete()` (incl. `DynamicResultProducer::Drop`) | `Finished` |
| `fail(reason)` (incl. `spawn` catching a panic) | `Failed(reason)` |

## Type bounds

| Type | `Send` | `Sync` | `Clone` |
|---|---|---|---|
| `DynamicResult<T>`         | yes if `T: Send` | yes if `T: Send + Sync` | **no** |
| `DynamicResultProducer<T>` | yes if `T: Send` | yes if `T: Send + Sync` | **no** |
| `Signal`                   | yes              | yes                    | yes     |

`T: Send + Sync + 'static` required. `get()` additionally requires `T: Clone`.
Multi-consumer? Wrap `Arc<DynamicResult<T>>`. Multi-producer is not supported.

## Private (not re-exported)

- `Inner<T>` — shared state behind `Arc`.
- `payload_to_string(Box<dyn Any + Send>) -> String` — best-effort panic
  payload decoder used by `spawn`.

## Dependencies

`std` only:

- `std::sync::{Arc, Condvar, Mutex, RwLock, atomic::AtomicBool}`
- `std::time::{Duration, Instant}`
- `std::thread::spawn`
- `std::panic::{catch_unwind, AssertUnwindSafe}`
- `std::any::Any`

No `tokio`, no `parking_lot`, no `crossbeam`, no `tracing`.

## Forbidden

- Third `pub fn` on `DynamicResult` beyond the four listed.
- Sixth `pub fn` on `DynamicResultProducer` beyond the five listed.
- `impl Clone` for either handle (use `Arc<DynamicResult<T>>`).
- `async fn` anywhere.
- `unsafe` (`#![forbid(unsafe_code)]` at crate root).
- Public access to `Inner<T>` or the underlying `Arc`.
- Re-exporting `std::sync::*` aliases or wrapping them in newtypes.
