# Core Module — Interface

> **Status:** **frozen** (since iter-014). Any breaking change to the surface

## Public

### `Config` (read-only snapshot)

```rust
pub struct Config { /* opaque */ }

impl Config {
    /// Load `product.json` from the given path. Defaults are filled in for any
    /// missing field. Returns `ConfigError::Io` if the file is unreadable and
    /// `ConfigError::Parse` if the JSON is malformed.
    pub fn load(path: impl AsRef<Path>) -> Result<Self, ConfigError>;

    /// Build with all defaults (no file required). Useful in tests.
    pub fn defaults() -> Self;

    pub fn dict_sqlite_path(&self) -> Option<&Path>;
    pub fn dict_seed_on_first_open(&self) -> bool;

    pub fn history_max_entries(&self) -> usize;
    pub fn notes_persist(&self) -> bool;
    pub fn lookup_prefer_notes_over_dict(&self) -> bool;
}
```

### `NoteStore`

```rust
pub struct NoteStore { /* HashMap<String, Note> */ }

impl NoteStore {
    pub fn new() -> Self;

    /// Insert or overwrite a Note. The `word` argument is lower-cased before storage.
    pub fn set(&mut self, word: &str, content: String);

    /// Remove a Note. Returns `true` if it existed, `false` if not (idempotent).
    pub fn remove(&mut self, word: &str) -> bool;

    /// Case-insensitive lookup.
    pub fn get(&self, word: &str) -> Option<&Note>;

    /// All notes, sorted by `word` ascending.
    pub fn list(&self) -> Vec<&Note>;

    pub fn len(&self) -> usize;
    pub fn is_empty(&self) -> bool;
}
```

### `HistoryStore`

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct HistoryEntry {
    pub word: String,
    pub at_unix_secs: i64,
}

pub struct HistoryStore { /* VecDeque<HistoryEntry> */ }

impl HistoryStore {
    /// Capacity is taken from `Config::history_max_entries`.
    pub fn with_capacity(max: usize) -> Self;

    /// Record a successful lookup. If the word is already at the front, only its
    /// timestamp is updated. Older duplicates are removed.
    pub fn record(&mut self, word: &str);

    /// Most recent first.
    pub fn recent(&self) -> &[HistoryEntry];

    pub fn clear(&mut self);
}
```

### `LookupService` and `LookupHit`

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum LookupHit {
    Note(Note),
    Dict(ee_dict::Entry),
}

#[derive(Debug, thiserror::Error)]
#[non_exhaustive]
pub enum LookupError {
    #[error("word not found")] NotFound,
    #[error("invalid input")]  InvalidInput,
    #[error("dictionary storage failure: {0}")] Storage(#[from] ee_dict::DictError),
}

pub struct LookupService { /* opaque */ }

impl LookupService {
    /// `prefer_notes_over_dict` from `Config` determines whether `query` checks
    /// the user's notes before falling back to the dictionary (default: true).
    pub fn new(prefer_notes_over_dict: bool) -> Self;

    /// Order: NoteStore (if preferred), then DictStore.
    pub fn query(
        &self,
        word: &str,
        notes: &NoteStore,
        dict: &ee_dict::DictStore,
    ) -> Result<LookupHit, LookupError>;
}
```

### Record models (`RecordModel`)

Polymorphic, internally-tagged (`record_type`) value stored in each
`storage_entries.value`. Providers return the serialized JSON; the UI
deserializes and renders per variant.

```rust
pub enum RecordModel {
    WordEn(WordEn),   // English headword, rich metadata (ipa, defs, inflections, examples)
    WordCn(WordCn),   // Chinese headword → most-frequent English equivalents
    Note(Note),
    History(History),
}

/// Chinese → English dictionary record (the `word_cn` dataset).
pub struct WordCn {
    pub word: String,          // canonical Chinese term (lookup key)
    pub english: Vec<String>,  // English equivalents, most-frequent first (≤ 10)
}
```

## Private

- The HashMap backing `NoteStore` is keyed by the lower-cased word; `get`/`remove`
  lower-case their argument before lookup.
- `HistoryStore` is `VecDeque<HistoryEntry>` so de-duplication of a recently-recorded
  word is O(n) over the deque (acceptable: `max_entries` is small).
- `AppState::submit` performs trimming and length validation before delegating to
  `LookupService::query`.
