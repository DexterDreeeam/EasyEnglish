# EasyEnglish

EasyEnglish 是一款轻量、易用的桌面词典，用于英语和其他语言之间的互译。

当前版本重点支持英语 ↔ 中文。更多语言库会在后续版本中加入。

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)

## 功能

- 应用启动时显示 First Auto Flyout。
- 通过全局快捷键快速唤出查询窗口。
- 支持英语到目标语言释义。
- 支持目标语言到英语候选词。
- 系统托盘支持显示窗口、开机启动和退出。

## 使用方式

- 启动 EasyEnglish。
- 使用 First Auto Flyout，或按 <kbd>Alt</kbd> + <kbd>`</kbd> 再次显示查询窗口。
- 输入英语或已支持的目标语言。
- 右键点击托盘图标可以显示查询窗口、切换开机启动或退出。
