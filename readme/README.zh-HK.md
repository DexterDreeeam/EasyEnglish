# EasyEnglish

EasyEnglish 是一款輕量、易用的桌面詞典，用於英語和其他語言之間的互譯。

按 <kbd>Alt</kbd> + <kbd>`</kbd> 顯示查詢視窗。

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)
