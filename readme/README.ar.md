# EasyEnglish

EasyEnglish هو قاموس مكتبي خفيف وسهل الاستخدام للترجمة بين الإنجليزية ولغات أخرى.

اضغط <kbd>Alt</kbd> + <kbd>`</kbd> لإظهار نافذة البحث.
