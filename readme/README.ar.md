# EasyEnglish

EasyEnglish هو قاموس مكتبي خفيف وسهل الاستخدام للترجمة بين الإنجليزية ولغات
أخرى.

يركز الإصدار الحالي على الإنجليزية ↔ الصينية المندرينية. ستتم إضافة مكتبات
لغات أخرى في الإصدارات القادمة.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)

## الميزات

- First Auto Flyout عند تشغيل التطبيق.
- بحث سريع عبر اختصار عام.
- تعريفات من الإنجليزية إلى اللغة الهدف.
- كلمات مرشحة من اللغة الهدف إلى الإنجليزية.
- عناصر تحكم من أيقونة النظام لإظهار النافذة، وتفعيل التشغيل عند بدء النظام،
  والخروج.

## الاستخدام

- شغّل EasyEnglish.
- استخدم First Auto Flyout أو اضغط <kbd>Alt</kbd> + <kbd>`</kbd> لإظهار نافذة
  البحث مرة أخرى.
- اكتب بالإنجليزية أو بلغة هدف مدعومة.
- انقر بزر الفأرة الأيمن على أيقونة النظام لإظهار النافذة أو تغيير التشغيل عند
  بدء النظام أو الخروج.
