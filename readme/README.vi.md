# EasyEnglish

EasyEnglish là một từ điển desktop nhẹ và dễ dùng để dịch giữa tiếng Anh và các
ngôn ngữ khác.

Bản phát hành hiện tại tập trung vào tiếng Anh ↔ tiếng Trung Quan thoại. Các thư
viện ngôn ngữ khác sẽ được bổ sung trong những bản sau.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)

## Tính năng

- First Auto Flyout khi ứng dụng khởi động.
- Tra cứu nhanh bằng phím tắt toàn cục.
- Định nghĩa từ tiếng Anh sang ngôn ngữ đích.
- Gợi ý từ ngôn ngữ đích sang tiếng Anh.
- Điều khiển qua khay hệ thống để hiển thị cửa sổ, bật khởi động cùng hệ thống,
  hoặc thoát.

## Cách dùng

- Khởi động EasyEnglish.
- Dùng First Auto Flyout hoặc nhấn <kbd>Alt</kbd> + <kbd>`</kbd> để mở lại cửa
  sổ tra cứu.
- Nhập tiếng Anh hoặc một ngôn ngữ đích được hỗ trợ.
- Nhấp chuột phải vào biểu tượng khay hệ thống để hiển thị cửa sổ, đổi tùy chọn
  khởi động cùng hệ thống, hoặc thoát.
