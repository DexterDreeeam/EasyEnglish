# EasyEnglish

EasyEnglish là một từ điển desktop nhẹ và dễ dùng để dịch giữa tiếng Anh và các ngôn ngữ khác.

Nhấn <kbd>Alt</kbd> + <kbd>`</kbd> để mở cửa sổ tra cứu.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)
