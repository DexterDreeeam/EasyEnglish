# EasyEnglish

EasyEnglish est un dictionnaire de bureau léger et facile à utiliser pour
traduire entre l'anglais et d'autres langues.

La version actuelle se concentre sur anglais ↔ chinois mandarin. D'autres
bibliothèques de langues seront ajoutées dans les prochaines versions.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)

## Fonctionnalités

- First Auto Flyout au lancement de l'application.
- Recherche rapide avec un raccourci global.
- Définitions de l'anglais vers la langue cible.
- Mots candidats de la langue cible vers l'anglais.
- Contrôles dans la zone de notification pour afficher la fenêtre, activer le
  lancement au démarrage ou quitter.

## Utilisation

- Lancez EasyEnglish.
- Utilisez le First Auto Flyout ou appuyez sur <kbd>Alt</kbd> + <kbd>`</kbd> pour
  afficher de nouveau la fenêtre.
- Saisissez de l'anglais ou une langue cible prise en charge.
- Faites un clic droit sur l'icône de la zone de notification pour afficher la
  fenêtre, changer le lancement au démarrage ou quitter.
