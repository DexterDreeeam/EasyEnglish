# EasyEnglish

EasyEnglish est un dictionnaire de bureau léger et facile à utiliser pour traduire entre l'anglais et d'autres langues.

Utilisez le First Auto Flyout ou appuyez sur <kbd>Alt</kbd> + <kbd>`</kbd> pour afficher de nouveau la fenêtre.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)
