# EasyEnglish

EasyEnglish は、英語と他の言語を相互に調べるための軽量で使いやすいデスクトップ辞書です。

First Auto Flyout を使うか、<kbd>Alt</kbd> + <kbd>`</kbd> で検索ウィンドウを再表示します。

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)
