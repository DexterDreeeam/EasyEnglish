# EasyEnglish

EasyEnglish는 영어와 다른 언어를 서로 찾아볼 수 있는 가볍고 사용하기 쉬운 데스크톱 사전입니다.

현재 릴리스는 영어 ↔ 중국어에 집중합니다. 더 많은 언어 라이브러리는 이후 버전에 추가될 예정입니다.

![EasyEnglish demo](../ee/docs/assets/easyenglish-demo.gif)

## 기능

- 앱 시작 시 First Auto Flyout 표시.
- 전역 단축키로 빠른 검색.
- 영어에서 대상 언어로 뜻풀이 표시.
- 대상 언어에서 영어 후보 단어 표시.
- 시스템 트레이에서 창 표시, 시작 시 실행, 종료 제어.

## 사용법

- EasyEnglish를 시작합니다.
- First Auto Flyout을 사용하거나 <kbd>Alt</kbd> + <kbd>`</kbd>를 눌러 검색 창을 다시 표시합니다.
- 영어 또는 지원되는 대상 언어를 입력합니다.
- 트레이 아이콘을 마우스 오른쪽 버튼으로 클릭하여 창 표시, 시작 시 실행, 종료를 제어합니다.
