# EasyEnglish

EasyEnglish는 영어와 다른 언어를 서로 찾아볼 수 있는 가볍고 사용하기 쉬운 데스크톱 사전입니다.

<kbd>Alt</kbd> + <kbd>`</kbd>를 눌러 검색 창을 표시합니다.
